# 🌌 About AstraPay

**AstraPay** is a **DeFi-inspired digital payment system** designed to demonstrate how modern financial platforms can deliver **fast, transparent, and secure transactions**.

The platform focuses on combining:

- ⚡ Fast digital payments  
- 🔐 Secure backend infrastructure  
- 🎨 Modern fintech UI  
- 🌐 API-based payment architecture  

Built during a **hackathon**, AstraPay explores how decentralized finance principles can improve traditional payment systems.



Example:
![image](https://pub-1407f82391df4ab1951418d04be76914.r2.dev/uploads/7203a686-c908-49fc-ae8c-ba376f6cd930.png)

# 🎯 Problem Statement

Modern payment systems suffer from several limitations:

• Slow transaction processing  
• High intermediary fees  
• Lack of transparency  
• Centralized control  

These limitations create inefficiencies in digital financial systems.

---

# 💡 Our Solution

AstraPay introduces a **modern payment experience inspired by decentralized finance (DeFi)**.

The platform focuses on:

- Fast transaction handling
- Transparent payment tracking
- Secure financial infrastructure
- API-first payment architecture

This makes AstraPay **ready for future blockchain integration**.

# ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/AstraPay.git
```

---

### 2️⃣ Navigate to Project

```bash
cd AstraPay
```

---

### 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

### 4️⃣ Run Backend Server

```bash
npm start
```

Server runs on:

```
http://localhost:5000
```

---

### 5️⃣ Open Frontend

Open in browser:

```
frontend/index.html
```

# 🔗 API Example

### Send Payment

```
POST /api/sendPayment
```

Request:

```json
{
  "sender": "UserA",
  "receiver": "UserB",
  "amount": 100
}
```

Response:

```json
{
  "status": "success",
  "transactionId": "TXN93843"
}
```
